# Projeto Site – Economia de Energia

## 📌 Descrição

Projeto web desenvolvido com foco em conscientização e economia de energia elétrica, apresentando informações, soluções e práticas voltadas à eficiência energética e sustentabilidade.

O objetivo do projeto é fornecer uma interface simples, moderna e acessível para demonstrar conceitos relacionados à redução do consumo energético e uso consciente de energia.

---

## 🌐 Acesse o Projeto

🔗 https://apolomarques.github.io/projeto-site-economia-de-energia/

## 🎯 Objetivos

- Incentivar o uso consciente de energia
- Demonstrar conceitos de eficiência energética
- Apresentar uma interface web moderna e responsiva
- Desenvolver habilidades em front-end e publicação web
